# Used Cars Data Preparation Project

## Project Overview

This project focuses on cleaning and preparing used car datasets for analysis.

## Objectives

* Handle missing values
* Remove duplicates
* Standardize data formats
* Perform exploratory data analysis
* Prepare data for visualization and modeling

## Tools Used

* Python
* Pandas
* NumPy
* Jupyter Notebook

## Skills Demonstrated

* Data Cleaning
* Data Preparation
* Exploratory Data Analysis (EDA)
* Data Transformation
* Data Validation
